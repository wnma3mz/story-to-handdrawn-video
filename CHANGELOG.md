## [1.0.0] - 2026-07-21

### Added

- Initial open-source release of the `story-to-handdrawn-video` skill.
- `SKILL.md` workflow contract: story-text beats, uploaded-page cropping,
  direct-cut and page-flip transitions, safe contained framing, silent MP4 output.
- Portable wrapper `scripts/run_story_video.py` with `STORY_VIDEO_PROJECT`
  discovery and upward directory walk.
- Agent display metadata in `agents/openai.yaml`.
